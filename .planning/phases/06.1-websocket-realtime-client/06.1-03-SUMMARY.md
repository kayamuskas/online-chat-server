# Phase 6.1 Plan 03 Summary

**One-liner:** The shared message timeline now autoscrolls intelligently and shows a floating `↓ новые сообщения` affordance when the user is reading older messages.

## What Was Built

Added scroll tracking, bottom-distance detection, and bottom-scroll callbacks to `MessageTimeline.tsx`. When the user is already near the bottom, new messages keep the view pinned. When the user has scrolled up, realtime arrivals no longer yank the viewport; the floating button appears instead.

The button clears parent unseen-message state once the user returns to the bottom, so both room and DM views can share the same behavior without duplicating scroll logic.
